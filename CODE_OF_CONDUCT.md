# Code of Conduct

## Our pledge

We are committed to making participation in this project respectful, inclusive, and harassment-free for everyone.

## Expected behavior

- Be respectful and constructive.
- Focus on ideas, not individuals.
- Assume positive intent and ask clarifying questions.
- Share feedback with actionable context.

## Unacceptable behavior

- Harassment, intimidation, or personal attacks
- Discriminatory or hateful language
- Doxxing or publishing private information
- Repeated disruptive behavior

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate this code of conduct.

## Reporting

Report concerns privately to project maintainers through your repository security/contact process.
